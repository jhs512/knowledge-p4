# 업무 지식 기반

work 네임스페이스의 최상위 기둥(pillar)입니다. 업무와 관련된 결정(decision), 개념(concept), 작업(task), 사실(fact) 노드들이 직접 또는 간접적으로 이 기둥에 연결됩니다.

새 노드를 만들 때 더 적절한 상위 노드가 없다면 이 기둥에 `supports` 또는 `part_of` 엣지로 연결하세요.
